## 🚀 Features

- **Amazon-style Header**
  - Brand logo
  - Top navigation menu with sections such as:
    - _All Articles_
    - _Today's Deals_
    - _Customer Service_
    - _Register_
    - _Gift Cards_
    - _Sell_

- **Holiday Promo Banner**
  - Promotional strip for Christmas gift cards
  - Main hero section with:
    - holiday headline
    - supporting text
    - strong visual hierarchy

- **Gift Cards Section**
  - Title area with CTA link
  - Product card gallery for gift inspiration
  - Clean card-based image layout

- **Promotional Categories**
  - Large featured category banners
  - Examples include:
    - _Play with gifts_
    - _The sound of happiness_
  - CTA links for category offers

- **Footer**
  - “Back to top” anchor link
  - Brand logo repeated in the footer
  - Simple structure inspired by e-commerce layouts

- **Educational Context**
  - Built as an early frontend exercise to practice static page composition, section hierarchy and commercial layout replication
